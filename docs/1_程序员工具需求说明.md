# 程序员工具需求说明

## 1. 概述

本文档描述一个面向程序员的在线工具平台，提供各类开发辅助工具、文档转换、加密解密、文本处理等功能。

---

## 2. 技术栈

### 2.1 前端技术栈
- **框架**: Vue 3 + Nuxt 4
- **UI组件库**: shadcn-vue
- **样式**: TailwindCSS
- **状态管理**: Pinia
- **图标库**: Lucide Vue Next

### 2.2 后端技术栈
- **框架**: Spring Boot 4
- **数据库**: PostgreSQL
- **缓存**: Redis
- **ORM**: JPA (Hibernate)
- **安全**: Spring Security + JWT

---

## 3. 功能模块

### 3.1 工具分类

平台工具分为以下六大类别：

#### 3.1.1 开发工具
- JSON格式化/压缩/校验
- XML格式化/压缩
- YAML/JSON互转
- 正则表达式测试
- SQL格式化
- 代码格式化（HTML/CSS/JS/Java等）
- Base64编码/解码
- URL编码/解码
- HTML实体编码/解码
- 时间戳转换
- 进制转换
- Cron表达式生成器

#### 3.1.2 图片工具
- 图片压缩
- 图片格式转换（PNG/JPG/WebP/SVG）
- 图片裁剪/旋转
- Base64图片转换
- 二维码生成/解析
- 条形码生成
- 图片水印
- 图片拼接

#### 3.1.3 文档转换
- PDF转Word/Excel/PPT
- Word转PDF
- Excel转PDF/CSV/JSON
- Markdown转HTML/PDF
- 图片转PDF
- 文档合并/拆分

#### 3.1.4 加密工具
- MD5加密
- SHA系列加密（SHA1/SHA256/SHA512）
- HMAC加密
- AES加密/解密
- RSA加密/解密
- Base64加密/解密
- 哈希值计算

#### 3.1.5 文本工具
- 文本对比
- 文本去重
- 文本排序
- 大小写转换
- 全角半角转换
- 简繁体转换
- 字数统计
- 文本加密
- 随机字符串生成

#### 3.1.6 数字工具
- 计算器
- 单位换算
- 汇率换算
- 贷款计算器
- 进制转换器
- 随机数生成器
- 密码生成器

### 3.2 首页模块

#### 3.2.1 页面布局
- **Hero区域**: 大标题 + 副标题 + 搜索框
- **分类快捷入口**: 横向排列的分类标签，带图标和Tooltip提示
- **Tab切换**: 热门推荐/最近访问/我的收藏/最新收录
- **分类区块**: 各分类工具展示，支持查看更多

#### 3.2.2 热门推荐
- 展示平台最热门的工具（按访问量排序）
- 默认显示前8个工具
- 卡片风格展示（图标+名称+描述+VIP标签）
- 4列网格布局（响应式：手机1列/平板2列/桌面4列）

#### 3.2.3 最近访问
- 展示用户最近使用过的工具
- 从 localStorage 读取访问记录
- 最多显示最近8个
- 空状态提示："暂无最近访问记录"

#### 3.2.4 我的收藏
- 展示用户收藏的工具
- 需要登录后显示
- 未登录状态提示登录
- 空状态提示："暂无收藏工具"

#### 3.2.5 最新收录
- 展示平台最新添加的工具
- 默认显示最近添加的8个

#### 3.2.6 搜索功能
- 全局搜索框，支持搜索工具名称
- 搜索后跳转到分类页并带搜索参数

### 3.6 权限系统

#### 3.6.1 角色定义

| 角色 | 说明 | 权限范围 |
|------|------|----------|
| 普通用户 | 注册后的默认角色 | 使用免费工具、收藏、评价 |

| 管理员 | 后台管理角色 | 管理后台所有功能 |

#### 3.6.2 管理员权限

**用户管理**

- 查看用户详情
- 禁用/启用用户账号
- 重置用户密码
- 修改用户会员等级

**工具管理**
- 添加新工具
- 编辑工具信息
- 启用/禁用工具
- 设置工具权限（免费/VIP）

**内容管理**
- 查看用户评价
- 删除违规评价
- 查看用户反馈
- 回复用户反馈

**系统管理**
- 平台运营统计查看
- 系统参数配置
- 日志查看

#### 3.7.1 页面布局

#### 3.7.3 评分评价系统（规划中）

**评分功能**
- 5星评分制，支持1-5星评分

- 每个用户对每个工具只能评分一次

**评价功能**
- 用户可撰写文字评价（最多200字）
- 评价为选填项，评分为必填项
- 支持点赞其他用户的评价
- 评价按时间倒序排列

**评分展示**
- 综合评分显示
- 评分分布图
- 评价总数显示
- 评价列表展示

### 3.8 会员与付费系统

#### 3.8.1 工具类型

**免费工具**
- 所有用户均可使用
- 部分功能可能受限

**VIP工具**
- 需要订阅会员
- 免费用户可预览功能但无法使用

#### 3.8.2 会员等级

| 会员等级 | 说明 | 价格 | 权益 |
|---------|------|------|------|
| 普通会员 | 注册后默认等级 | 免费 | 使用免费工具 |
| 包月会员 | 按月订阅 | ¥19/月 | 使用所有付费工具 |
| 包季会员 | 按季订阅 | ¥49/季 | 比包月更优惠 |
| 包年会员 | 按年订阅 | ¥99/年 | 最大优惠力度 |
| 永久会员 | 一次性购买 | ¥299 | 永久享受所有会员权益 |

#### 3.8.3 会员权益对比

| 功能 | 普通 | 包月 | 包季 | 包年 | 永久 |
|------|------|------|------|------|------|
| 基础工具使用 | ✓ | ✓ | ✓ | ✓ | ✓ |
| 高级工具使用 | ✗ | ✓ | ✓ | ✓ | ✓ |
| 单文件大小限制 | 1MB | 10MB | 10MB | 50MB | 100MB |
| 每日使用次数 | 10次 | 无限 | 无限 | 无限 | 无限 |
| 广告显示 | ✓ | ✗ | ✗ | ✗ | ✗ |
| 云存储空间 | - | - | - | 5GB | 10GB |
| 客服支持 | 普通 | 优先 | 优先 | 优先 | VIP |
| 新功能抢先体验 | ✗ | ✗ | ✓ | ✓ | ✓ |

#### 3.8.4 定价页功能

**会员套餐展示**
- 卡片式展示各等级会员套餐
- 突出显示推荐套餐（包年）
- FAQ常见问题解答

**权益对比表**
- 详细对比各等级会员权益

## 4. 页面结构

### 4.1 前端页面

```
├── 首页 (/)
│   ├── Hero区域（搜索框）
│   ├── 分类快捷入口
│   ├── Tab切换（热门/最近/收藏/最新）
│   └── 分类工具展示
│
├── 分类页 (/category/:id)
│   ├── 左侧筛选栏
│   └── 右侧工具列表
│
├── 工具详情 (/tool/:id)
│   └── 具体工具的使用界面
│
├── 定价 (/pricing)
│   ├── 会员套餐对比
│   └── FAQ
│
├── 排行榜 (/ranking)
│   └── 日/周/月/总榜
│
├── 关于 (/about)
│   └── 平台介绍
│
├── 联系 (/contact)
│   └── 联系方式和表单
│
├── 帮助 (/help)
│   └── FAQ
│
├── 反馈 (/feedback)
│   └── 问题反馈表单
│
├── 隐私政策 (/privacy)
│   └── 隐私政策内容
│
├── 使用条款 (/terms)
│   └── 使用条款内容
│
├── 登录 (/login)
│   └── 邮箱/第三方登录
│
└── 注册 (/register)
    └── 邮箱/第三方注册
```

### 4.2 导航栏结构

**主导航（顶部导航栏）**
- Logo + 品牌名（秋云工具）
- 首页
- 分类
- 定价
- 排行榜

**右侧功能区**
- 主题切换按钮（🌙/☀️）
- 未登录：登录/注册按钮
- 已登录：用户头像下拉菜单

**导航栏规范**
- 高度：滚动前64px，滚动后56px
- 背景：半透明毛玻璃效果
- 固定在顶部
- 滚动时显示底部边框

### 4.3 页脚结构

**页脚内容**
- 品牌Logo和简介
- 工具分类链接
- 关于链接
- 支持链接
- 社交媒体图标（GitHub、Bilibili、邮箱）
- Copyright（动态年份）
- Made with ❤️ 标语

**悬浮反馈按钮**
- 固定在右下角
- 点击打开反馈弹窗
