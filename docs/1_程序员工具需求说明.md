# 程序员工具需求说明

## 1. 概述

本文档描述一个面向程序员的在线工具平台，提供各类开发辅助工具、文档转换、加密解密、文本处理等功能。

---

## 2. 技术栈

### 2.1 前端技术栈
- **框架**: Vue 3 + Nuxt 4
- **UI组件库**: shadcn-vue
- **样式**: TailwindCSS
- **状态管理**: Pinia

### 2.2 后端技术栈
- **框架**: Spring Boot 4
- **数据库**: PostgreSQL
- **缓存**: Redis
- **ORM**: JPA (Hibernate)
- **安全**: Spring Security + JWT

---

### 3.7 会员与付费系统

#### 3.7.1 工具类型

**免费工具**
- 所有用户均可使用
- 部分功能可能受限（如文件大小限制、使用次数限制）

**付费工具**
- 需要订阅会员或购买使用次数
- 免费用户可预览功能但无法使用

#### 3.7.2 会员等级

| 会员等级 | 说明 | 价格 | 权益 |
|---------|------|------|------|
| 普通会员 | 注册后默认等级 | 免费 | 使用免费工具，付费工具需单独购买 |
| 包月会员 | 按月订阅 | ¥19/月 | 使用所有付费工具，享受会员专属功能 |
| 包季会员 | 按季订阅 | ¥49/季 | 比包月更优惠，权益同包月 |
| 包年会员 | 按年订阅 | ¥99/年 | 最大优惠力度，权益同包月+去广告+云存储 |
| 永久会员 | 一次性购买 | ¥299 | 永久享受所有会员权益+终身更新 |

#### 3.7.3 会员权益对比

| 功能 | 普通 | 包月 | 包年 | 永久 |
|------|------|------|------|------|
| 基础工具使用 | ✓ | ✓ | ✓ | ✓ |
| 高级工具使用 | ✗ | ✓ | ✓ | ✓ |
| 单文件大小限制 | 1MB | 10MB | 50MB | 100MB |
| 每日使用次数 | 10次 | 无限 | 无限 | 无限 |
| 广告显示 | ✓ | ✓ | ✗ | ✗ |
| 云存储空间 | - | - | 5GB | 10GB |
| 客服支持 | 普通 | 优先 | 优先 | VIP |
| 新功能抢先体验 | ✗ | ✗ | ✓ | ✓ |

#### 3.7.4 充值与购买

**会员订阅**
- 支持支付宝/微信支付
- 支持自动续费（包月/包季/包年）
- 订阅到期提醒

**按次购买**
- 普通会员可充值余额
- 使用余额购买特定工具的使用次数
- 支持购买单次或多次套餐

**积分系统**
- 每日签到获得积分
- 分享工具获得积分
- 积分可抵扣部分费用

#### 3.7.5 定价页功能

**会员套餐展示**
- 卡片式展示各等级会员套餐
- 突出显示推荐套餐（如包年）
- 显示原价与优惠价对比

**权益对比表**
- 详细对比各等级会员权益
- 功能权限对比（工具使用/文件大小/使用次数）
- 增值服务对比（客服支持/云存储/去广告）

**支付流程**
- 选择套餐 → 选择支付方式 → 确认支付 → 支付成功
- 支持支付宝/微信支付
- 支持优惠码/积分抵扣
- 显示支付安全提示

**常见问题**
- 会员权益说明
- 退款政策
- 自动续费说明
- 升级/降级规则

### 3.9 问题反馈系统

#### 3.8.1 反馈入口
- **页脚反馈链接**: 所有页面页脚显示"问题反馈"链接
- **悬浮反馈按钮**: 页面右下角固定显示反馈按钮（💬 反馈）
- **工具详情页反馈**: 工具操作区下方显示"🐛 遇到问题？反馈给我们"链接

#### 3.8.2 反馈类型
- 🐛 功能异常: 工具无法正常使用、报错等
- 📝 功能建议: 新功能需求、改进建议
- 📱 界面问题: UI显示异常、交互问题
- ❓ 其他问题: 不属于以上类型的问题

#### 3.8.3 反馈内容
- 反馈类型选择
- 相关工具（选填，自动关联当前工具）
- 问题描述（多行文本，必填）
- 联系方式（选填，邮箱或手机号）

#### 3.8.4 反馈处理
- 用户提交后显示成功提示
- 后台管理可查看所有反馈
- 支持标记处理状态（待处理/处理中/已解决）
- 支持回复用户（通过邮件或站内信）

│
├── 登录/注册 (/auth/login, /auth/register)
│

## 7. 统计功能

### 7.1 平台运营统计（仅管理员可见）

#### 7.1.1 用户统计
- **注册用户统计**
  - 总注册用户数
  - 日/周/月新增用户数
  - 用户增长趋势图
  - 用户来源分布（直接访问/搜索引擎/社交媒体/推广链接）

- **活跃用户统计**
  - 日活跃用户（DAU）
  - 周活跃用户（WAU）
  - 月活跃用户（MAU）
  - 用户留存率（次日/7日/30日留存）

- **会员统计**
  - 各等级会员数量分布
  - 会员转化率
  - 会员续费率
  - 会员收入统计

#### 7.1.2 工具使用统计
- **整体使用统计**
  - 工具总使用次数
  - 日/周/月使用次数趋势
  - 人均使用次数

- **分类统计**
  - 各分类工具使用占比
  - 分类使用趋势对比

- **单个工具统计**
  - 每个工具的使用次数
  - 工具使用趋势
  - 工具平均使用时长
  - 工具错误率/失败率

#### 7.1.3 收入统计
- **收入概览**
  - 总收入金额
  - 日/周/月收入趋势
  - 收入构成（会员订阅/按次购买/积分兑换）

- **订单统计**
