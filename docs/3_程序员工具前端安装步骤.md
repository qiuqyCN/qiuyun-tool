### 前端项目安装步骤
参考链接: https://www.shadcn-vue.com/docs/installation/nuxt

1. 创建项目
```bash
npm create nuxt@latest
npm install -D typescript
```

2. 安装 Tailwind CSS
```bash
npm install tailwindcss @tailwindcss/vite -D
```

3. 配置 Tailwind CSS
app/assets/css/tailwind.css
```bash
@import "tailwindcss";
```


4. 安装 Color Mode 模块
```bash
npm install @nuxtjs/color-mode
```
