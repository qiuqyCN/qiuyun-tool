# 程序员工具UI设计图

## 设计规范 (shadcn-vue 风格)

### 色彩系统

#### 亮色模式 (Light Mode)
```
主色调 (Zinc 灰度基调):
- Background: #FFFFFF (白色背景)
- Foreground: #09090B (近黑色文字)
- Primary: #18181B (深灰主色)
- Primary Foreground: #FAFAFA (主色上的文字)

次要色:
- Secondary: #F4F4F5 (浅灰背景)
- Secondary Foreground: #18181B
- Muted: #F4F4F5
- Muted Foreground: #71717A (次要文字)

强调色:
- Accent: #F4F4F5
- Accent Foreground: #18181B
- Destructive: #EF4444 (红色)
- Destructive Foreground: #FAFAFA

边框与输入:
- Border: #E4E4E7
- Input: #E4E4E7
- Ring: #18181B (聚焦环)
```

#### 暗色模式 (Dark Mode)
```
主色调:
- Background: #09090B (近黑色背景)
- Foreground: #FAFAFA (白色文字)
- Primary: #FAFAFA (白色主色)
- Primary Foreground: #18181B (主色上的文字)

次要色:
- Secondary: #27272A (深灰背景)
- Secondary Foreground: #FAFAFA
- Muted: #27272A
- Muted Foreground: #A1A1AA (次要文字)

强调色:
- Accent: #27272A
- Accent Foreground: #FAFAFA
- Destructive: #EF4444 (红色)
- Destructive Foreground: #FAFAFA

边框与输入:
- Border: #27272A
- Input: #27272A
- Ring: #D4D4D8 (聚焦环)
```

#### 圆角
```
- Radius: 0.5rem (8px)
- Large: 0.75rem (12px)
- XL: 1rem (16px)
```

### 字体规范
```
字体族：Inter, system-ui, -apple-system, sans-serif

标题：
- H1: 2.25rem/2.5rem (36px), font-weight: 700
- H2: 1.875rem/2.25rem (30px), font-weight: 600
- H3: 1.5rem/2rem (24px), font-weight: 600
- H4: 1.25rem/1.75rem (20px), font-weight: 600

正文：
- Large: 1.125rem/1.75rem (18px)
- Base: 1rem/1.5rem (16px)
- Small: 0.875rem/1.25rem (14px)
- XSmall: 0.75rem/1rem (12px)
```

### 间距系统
```
基础单位: 0.25rem (4px)
1: 0.25rem (4px)
2: 0.5rem (8px)
3: 0.75rem (12px)
4: 1rem (16px)
5: 1.25rem (20px)
6: 1.5rem (24px)
8: 2rem (32px)
10: 2.5rem (40px)

```

### 组件风格
```
按钮:
- Default: bg-primary text-primary-foreground hover:bg-primary/90
- Secondary: bg-secondary text-secondary-foreground hover:bg-secondary/80
- Outline: border border-input bg-background hover:bg-accent
- Ghost: hover:bg-accent hover:text-accent-foreground
```
